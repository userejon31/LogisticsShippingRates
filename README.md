# Calculador de Interés Simple

Este proyecto incluye un script para calcular el interés simple usando la fórmula:

> Interés Simple = Principal × Tasa de interés × Tiempo

- **Principal:** Monto inicial.
- **Tasa de interés:** Debe ingresarse como decimal (por ejemplo, 5% = 0.05).
- **Tiempo:** Número de períodos (años, meses, etc.).

## Ejemplo de uso

Suponga que ingresa:
- Principal: 1000
- Tasa de interés: 0.05
- Tiempo: 2

El interés simple será: 1000 × 0.05 × 2 = 100

## Cómo ejecutar

Desde tu terminal:

```bash
bash simple-interest.sh 1000 0.05 2
```
